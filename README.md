# Todo Application

My first React JS project — a simple to-do list app built while learning React, featuring full CRUD operations, local storage persistence, and motivational quotes on refresh.

**Live:** [todo-list-app-12.netlify.app](https://todo-list-app-12.netlify.app/)

## Features
- Create, read, update, and delete tasks
- Tasks saved to local storage
- Motivational quote on every refresh
- Fully responsive single-page application
